# Quray Static Website

This project is now a plain static site:

- `index.html` (home + hero)
- `about.html`
- `contact.html`
- `early-access.html`
- `styles.css`
- `main.js`

## Assets

- Logo: `assets-images/Quray_logo.svg`
- Hero video: `video/quray-hero.mp4`

## Run locally

Use any static server. Example:

```bash
python3 -m http.server 8000
```

Then open:

`http://localhost:8000`
